# Interior_Sharing_Site
3학년 1학기 웹 프로그래밍 팀 프로젝트

## 1. 개요
웹 프로그래밍이라는 학교 강의 때 2인 팀 프로젝트로 제작했던 사이트입니다. 
자신만의 개인적인 장소의 인테리어 구조와 소품 이러한 아이디어를 공유하는 사이트를 구현하고자 하였습니다.

## 2. 개발 기간
2018.03 - 2018.06

## 3. 개발 환경
Visual Code

## 4. 데이터베이스

## 5. 서버
